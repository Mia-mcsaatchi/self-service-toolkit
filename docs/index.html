What this tool does

The Self-Service Toolkit takes any spreadsheet — a Brandwatch export, a social listening dataset, a survey response file — and adds new AI-generated columns to it. You define what each new column should contain in plain English, and the tool runs every row through GPT-4o-mini to fill it in.

The result is your original data with extra columns appended, ready to download as CSV or Excel.

No setup neededThe tool runs in your browser and connects to a shared backend. You do not need an OpenAI account, API key, or any software installed.

Upload a CSV or Excel file (.xlsx) using the file picker. For Excel files with multiple tabs, select the worksheet you want. Then click Upload to backend.

Step 2 — Configure fields

Click + Add field to create a field card. Each card produces one or more new output columns.

Single vs Cluster

Single produces one output column. Use for one piece of information per row (sentiment, language, a yes/no flag).

Cluster produces multiple output columns from a single AI call — faster and cheaper. Use when several outputs are related and can be described together (topic + sentiment + key quote).

Default vs Conditional

Default mode runs the same prompt on every row. Use for universal tasks like translation or sentiment classification.

Conditional mode runs different prompts depending on the value of another column. Use when the question you want to ask depends on what is already in the data.

How branches workBranches are evaluated top to bottom. The first branch where all conditions match is the one that runs. If nothing matches, the field writes "n/a" for that row and no AI call is made.
Conditional requires categorical columnsThe condition column must have 30 or fewer unique values. Free-text columns like post_text are greyed out in the dropdown. Columns like overall_sentiment, language, or country_code work well.

Using the prompt bank

Every field has a prompt bank dropdown above the prompt textarea. Selecting a preset fills in a ready-to-use prompt. You can edit it freely after selecting.

Available presets: sentiment classification, topic categorisation, full tagging cluster, language detection, translation to English, purchase intent, complaint detection, brand mention tagging, one-sentence summary, urgency scoring, competitor comparison signal.

Step 3 — Run and export

Once fields are saved, Step 3 unlocks. Set Max rows to a small number (e.g. 10) for a test run before processing the full dataset. Leave at 0 to process everything. Click Run pipeline and wait for the progress bar to complete.

AND within a branch, OR across branchesAll conditions in one branch must match (AND). Branches are tried in order and the first full match wins (OR). Multiple values within one condition mean any of those values match (OR within values).

Tips and edge cases

Test with a small batch first

Set Max rows to 10 before running the full dataset. Check the output looks right, then set it back to 0. This saves time and API cost if the prompt needs tweaking.

The "unsure" token

If the AI cannot determine an answer — because the text is too short, ambiguous, or off-topic — it writes unsure. This is intentional. Filter for unsure in your exported file to find rows that need manual review.

The "n/a" value in conditional fields

n/a means no branch matched — no AI call was made. This is different from unsure, which means an AI call was made but the model could not determine the answer. Both are safe to filter out in your analysis.

Choose reads from carefully

Only tick columns that are genuinely useful context for the prompt. Including too many columns, especially long free-text fields, inflates the prompt and can confuse the model. One or two source columns is usually enough.

Cluster fields and JSON format

When using Cluster mode, the AI returns all output columns as a JSON object. Make sure your prompt explicitly lists the JSON keys and what each should contain. The prompt bank cluster presets do this correctly — use them as a starting point.

Conditional mode is greyed out

If the Conditional button appears greyed out with a tooltip, it means your data has no categorical columns — all columns have more than 30 unique values. Check that columns like overall_sentiment, language, or country_code are included in your file.
